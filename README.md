## 🚀 Featured Projects

### 🌽 [Crop Yield Prediction — Nepal](https://github.com/pratik-9051/crop-yield-prediction-nepal)
> Predicting district-level crop yields across Nepal using meteorological and agricultural data

- **Data:** NASA POWER + Ministry of Agriculture & Livestock Development (MoALD), Nepal
- **Models:** XGBoost, Random Forest, SVR, Ridge, Lasso (XGBoost best performer)
- **Pipeline:** Data collection → preprocessing → feature engineering → model training → deployment
- **Deployed:** Live Streamlit web application
- **Stack:** Python · XGBoost · Scikit-learn · Streamlit · Pandas
